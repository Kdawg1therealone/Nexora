# Nexora Portfolio Website

A complete responsive portfolio website for Nexora.

## Included
- Home / hero section
- Project pages for Momentum and Vault
- Services
- Real code showcase
- About Nexora
- Reviews
- Contact / Hire Us form
- FAQ
- Mobile navigation and scroll animations

## Run locally
Open `index.html` in a browser, or use the VS Code **Live Server** extension.

## Deploy with Vercel
1. Upload this folder to a new GitHub repository.
2. In Vercel, choose **Add New → Project**.
3. Import the repository.
4. Leave the framework preset as **Other**.
5. Click **Deploy**.

## Important edit
Open `script.js` and replace:
`your-email@example.com`
with the email address you want project requests sent to.

## Images
The Momentum and Vault visuals were created during the Nexora design process. The code image is based on real Flutter architecture from Momentum.


## Portfolio Pages
- `portfolio.html` — Complete portfolio gallery
- `project-shining-development.html` — Shining Development Workspace case study
- Project images and code visuals are in `assets/projects/shining-development/`

## Site Structure
- `portfolio.html` — Smaller scripts, models, systems and design work
- `projects.html` — Large applications, websites and major case studies
- `testimonials.html` — Full client testimonials
- `portfolio-fruit-pack.html` — Stylized Fruit Asset Pack portfolio item


## Important Upload Instructions

Upload the complete extracted folder, including the `assets` folder.

Do not upload only the HTML files. The website pages rely on the image files stored inside `assets`.

This version uses simplified, flat image filenames to reduce path issues on GitHub and Vercel.
